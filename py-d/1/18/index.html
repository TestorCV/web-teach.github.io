Що ми зробили за два заняття?


За заняття ми ознайомились з шаблонізотором Jinja та зробили основні шаблони, де все буде відображатись.

Можливості Jinja


  • {{ name }} - вставлення змінної.
  • {{ super() }} - функція випується в блок, для того, щоб зберегти вміст блоку не замінюючи, а доповнюючи.
  • {% include 'file.html' %} - підключити якийсь файл.
  • {% for w in www %}{% endfor %} - цикл for.
  • {% if smth == smth %}{% endif %} - умовний оператор if.
  • {% macro name(attribute) %}{% endmacro %} - Визначення функції, потім викликається, як змінна.
  • {% block name %}{% endblock %} - створення місця, куди буде підключатись інший файл.
  • {% extends 'file.html' %}
    {% block name %}{% endblock %}     - підключення головного файлу, та вставлення блоку в нього.
