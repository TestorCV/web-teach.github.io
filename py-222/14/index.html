Мова SQL


  • SELECT pass - Вибрати колонку.

  • FROM table - З таблиці.

  • SELECT * FROM table - Повний вираз. Означає - ВИБРАТИ УСЕ З ТАБЛИЦІ.

  • INSERT INTO table(pass, name) - Вставити в таблицю в рядки pass та name.

  • VALUES(%s,%s) - Значення. %s - означає, що замість них, ми будемо вставляти якісь данні з пайтону. Просто змінну, якщо треба передати одне значення. Кортеж, якщо треба передати декілька значень.

  • INSERT INTO table(pass, name) VALUES(%s,%s) - Повний вираз. Означає - ВСТАВИТИ В ТАБЛИЦЮ В РЯДКИ pass та name ЗНАЧЕННЯ %s та %s, які потім будуть передаватись кодом.
