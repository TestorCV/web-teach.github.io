Список уроків Тема 20. Форми. ВІДЕО
ЗДАТИ ВИКОНАНЕ ЗАВДАННЯ

План:
  1. Створюємо форму. Тег form.
  2. Додаємо звичайне текстове поле. Тег input і атрибут type = "text".
  3. Підказка в полі. Атбрібут placeholder.
  4. Задаємо мітку для елемента. Тег label і атрибут for.
  5. Додаємо поле для введення пароля. Атрибут type і його значення password.
  6. Вставляємо заблоковане поле за допомогою атрибута disabled.
  7. Додаємо список, що розкривається. Теги select і option.
  8. Вставляємо елементи «прапорці» за допомогою атрибута type і його значення checkbox.
  9. Додаємо елементи «перемикачі» за допомогою атрибута type і його значення radio.
  10. Елемент для завантаження файлу. Атрибут type і його значення file.
  11. Створюємо поле для багаторядкового введення тексту. Тег textarea.
  12. Додаємо кнопку для очищення форми. Тег button і атрибут type = reset.
  13. Створюємо кнопку для «відправки» форми. Атрибут type і його значення submit.
  14. Задаємо дію після заповнення форми. Атрибут action.
  15. Як задати стиль для певного типу поля?
